# Add a worker account

A worker account is just a separate Claude Code config directory
(`~/.claude-<name>`) with its own login.

1. Run **Add Worker Account** and pick a name (e.g. `w1`) and a default model.
2. A terminal opens running Claude Code under that directory — sign in **once**
   with the Claude account you want this worker to use.
3. Repeat for as many accounts as you have.

Already have `~/.claude-*` directories from other tools? Use
**Import Existing Claude Config Directories** instead — no re-login needed.
